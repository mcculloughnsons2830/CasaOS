# Ms'FitZ Studio — Brand Activation Package

The complete, ready-to-deploy activation system for Ms'FitZ Studio: the mascot-driven,
urban-misfit content brand that sits beside ÆNIGMA in the family
(see `../BRAND_GUIDE.md` §1a for the brand architecture).

**Core identity in one line:** *Misfits who see the Signal.*

## What's in this package

| File | What it is | Use it when |
|---|---|---|
| `BRAND_BIBLE.md` | The full brand bible: mascot family, palette, voice, visual rules, narrative arcs, naming conventions | Before creating *anything* — this is the source of truth |
| `SHORTS_PACK.md` | Shorts script templates + the full 20-caption pack + SEO strategy | Producing/scheduling Shorts, TikToks, Reels |
| `CONTENT_CALENDAR_30DAY.md` | The 30-day "Misfit Signal Rising" launch calendar | Planning the first month, then repeating the cadence |
| `MERCH_CATALOG.md` | Product line, mockup descriptions, BigCartel store copy, bundles, pricing | Setting up the store and product pages |
| `LEAD_MAGNET_7_SIGNS.md` | "7 Signs Your Outsider Season Is Ending" — full written content, ready for PDF layout | Building the email-capture lead magnet |
| `PROMPT_LIBRARY.md` | Tested image/video generation prompts for every mascot and scene type | Generating new on-brand visuals |

## Launch sequence (Week 1)

1. **Day 1–2:** Upload first 3 Shorts using existing rabbit/moth videos + captions from `SHORTS_PACK.md`.
2. **Day 3–4:** Lay out `LEAD_MAGNET_7_SIGNS.md` as a PDF (Canva, dark template per the bible) and put it behind an email capture in the link-in-bio.
3. **Day 5–7:** Open the BigCartel store with the 3 core products + 1 bundle from `MERCH_CATALOG.md`.
4. **Ongoing:** Follow `CONTENT_CALENDAR_30DAY.md` — 5 Shorts/week + 1 long-form/week. Weekly performance review; double down on the top-retaining pillar.

## Relationship to the ÆNIGMA brand

- ÆNIGMA is the mirror (the product/agent). Ms'FitZ Studio is the creative house and the street-level voice.
- Shared palette core: Void Black, Signal Teal, Bone White (hexes in `BRAND_BIBLE.md`).
- Studio adds its own accents (Luminous Orange/Gold eyes, Neon Pink night energy) that ÆNIGMA does not use.
- Cross-promotion flows one way on-screen: Studio Shorts may end with the ÆNIGMA credit line; ÆNIGMA surfaces credit the Studio per the standard credit line in `../BRAND_GUIDE.md`.
