Script templates, the full 20-caption pack, and the SEO layer for YouTube Shorts /
TikTok / Reels. Format: vertical 9:16, 8–15 second loops built from the existing
rabbit/moth footage, lo-fi/hip-hop or rain-ambient audio, burned overlay text in
Bone White, `@MsfitZ-Vibe` watermark bottom-center.

Voice rules: see `BRAND_BIBLE.md` §5. One glow color per piece.

---

## 1. The loop formula

Every Short is the same machine:

1. **0–1s — pattern break:** character already in frame, rain falling, eyes dark.
2. **1–3s — hook text appears** (the caption's first line).
3. **3–8s — the beat:** subtle motion only — head nod, breath fog, rain shift.
4. **8–12s — the turn:** eye glow pulses on, synced to the beat drop. Second text line.
5. **12–15s — CTA + end-card:** "Drop 🔥 if you feel it." → Subscribe + watermark.

The loop should be seamless — glow fades as rain resets, so rewatches feel intentional.
Target: 40%+ average view duration; the glow-pulse *is* the retention hook.
